1. Definición y operaciones con vectores

Vector, componentes, módulo, suma, producto por escalar y descomposición canónica.

Primera sección desarrollada

Vector

Un vector puede representarse como un segmento orientado. Queda determinado por módulo, dirección y sentido.

En el plano, si el vector parte del origen y llega al punto (u₁,u₂), se escribe:

𝐮 = (u₁, u₂)
‖𝐮‖ = √(u₁² + u₂²)

Suma de vectores

La suma de vectores puede interpretarse geométricamente mediante la regla del triángulo o la regla del paralelogramo.

𝐰 = 𝐮 + 𝐯
𝐮 + 𝐯 = (u₁ + v₁, u₂ + v₂)

Producto por escalar

Si c es un escalar, entonces c𝐮 es un vector en la misma dirección de 𝐮.

  • Si c > 0, conserva el sentido.
  • Si c < 0, invierte el sentido.
  • Si c = 0, se obtiene el vector nulo.
c𝐮 = (cu₁, cu₂)
‖c𝐮‖ = |c| ‖𝐮‖

Descomposición canónica

En el espacio, usando los versores 𝐢, 𝐣 y 𝐤, todo vector puede escribirse como:

𝐮 = (u₁,u₂,u₃) = u₁𝐢 + u₂𝐣 + u₃𝐤

2. Producto escalar

Definición geométrica, componentes, signo, ortogonalidad, ángulo, proyección y aplicaciones.

Sección desarrollada

Definición geométrica

El producto escalar entre dos vectores es una operación que devuelve un número real. Si θ es el ángulo entre 𝐮 y 𝐯, se define:

𝐮 · 𝐯 = ‖𝐮‖ ‖𝐯‖ cos(θ)

Esta expresión permite relacionar el valor del producto escalar con la orientación relativa de los vectores.

Producto escalar por componentes

En coordenadas cartesianas, el producto escalar se calcula multiplicando componentes homónimas y sumando los resultados.

𝐮 · 𝐯 = u₁v₁ + u₂v₂
𝐮 · 𝐯 = u₁v₁ + u₂v₂ + u₃v₃

Signo y ortogonalidad

  • Si θ es agudo, entonces 𝐮 · 𝐯 > 0.
  • Si θ es recto, entonces 𝐮 · 𝐯 = 0.
  • Si θ es obtuso, entonces 𝐮 · 𝐯 < 0.
𝐮 ⟂ 𝐯 ⇔ 𝐮 · 𝐯 = 0

Ángulo y proyección

El ángulo entre dos vectores no nulos se obtiene de:

cos(θ) = (𝐮 · 𝐯) / (‖𝐮‖ ‖𝐯‖)

La proyección de 𝐯 sobre 𝐮 es:

proy𝐮(𝐯) = ((𝐮 · 𝐯) / ‖𝐮‖²) 𝐮

3. Producto vectorial

Vector perpendicular, regla de la mano derecha, cálculo por determinante, área y aplicaciones.

Sección desarrollada

Definición geométrica

El producto vectorial entre dos vectores de ℝ³ da como resultado otro vector. Si 𝐰 = 𝐮 × 𝐯, entonces 𝐰 es perpendicular al plano determinado por 𝐮 y 𝐯.

𝐰 = 𝐮 × 𝐯
‖𝐮 × 𝐯‖ = ‖𝐮‖ ‖𝐯‖ sen(θ)

Dirección y sentido

La dirección de 𝐮 × 𝐯 es perpendicular a 𝐮 y a 𝐯. El sentido se determina con la regla de la mano derecha o regla del sacacorchos.

Si se invierte el orden, cambia el sentido:

𝐮 × 𝐯 = −(𝐯 × 𝐮)

Cálculo por componentes

Si 𝐮=(u₁,u₂,u₃) y 𝐯=(v₁,v₂,v₃), entonces:

𝐮 × 𝐯 = (u₂v₃−u₃v₂, u₃v₁−u₁v₃, u₁v₂−u₂v₁)

También puede calcularse mediante el determinante simbólico con 𝐢, 𝐣 y 𝐤.

Área y aplicaciones

El módulo del producto vectorial representa el área del paralelogramo generado por 𝐮 y 𝐯.

A = ‖𝐮 × 𝐯‖

También se utiliza para calcular momentos de fuerzas y distancias geométricas.

𝐌 = 𝐫 × 𝐅

4. Producto mixto

Producto mixto, volumen del paralelepípedo, coplanaridad, determinante y regla cíclica.

Sección desarrollada

Definición

El producto mixto combina el producto vectorial y el producto escalar. Dados tres vectores de ℝ³, se define:

[𝐮,𝐯,𝐰] = 𝐮 · (𝐯 × 𝐰)

El resultado del producto mixto es un número real.

Interpretación geométrica

El valor absoluto del producto mixto representa el volumen del paralelepípedo generado por 𝐮, 𝐯 y 𝐰.

V = |𝐮 · (𝐯 × 𝐰)|

El producto 𝐯×𝐰 se asocia con el área de la base y el producto escalar con 𝐮 incorpora la altura orientada.

Coplanaridad

Si los tres vectores son coplanares, el volumen del paralelepípedo es cero. Por eso:

𝐮 · (𝐯 × 𝐰) = 0 ⇔ 𝐮, 𝐯 y 𝐰 son coplanares

Determinante y regla cíclica

El producto mixto puede calcularse con un determinante de orden 3:

𝐮 · (𝐯 × 𝐰) = det[𝐮;𝐯;𝐰]

Las permutaciones cíclicas conservan el valor:

𝐮·(𝐯×𝐰) = 𝐯·(𝐰×𝐮) = 𝐰·(𝐮×𝐯)

Distancia dirigida, distancia entre puntos y división de segmentos

Del eje coordenado al plano cartesiano.

Teoría · actividades · autoevaluación

Distancia dirigida

En un eje coordenado, el segmento dirigido de \(P_1\) a \(P_2\) conserva signo.

P₁P₂ = x₂ − x₁

Al invertir el sentido, cambia el signo: \(P₂P₁=x₁-x₂\).

Distancia entre dos puntos

En el plano, para \(P_1(x_1,y_1)\) y \(P_2(x_2,y_2)\):

d(P₁,P₂)=√((x₂−x₁)²+(y₂−y₁)²)

Punto medio

El punto medio \(M\) de un segmento es:

M=((x₁+x₂)/2,(y₁+y₂)/2)

División de un segmento

Si \(P\) divide a \(P_1P_2\) en razón \(r=P_1P/PP_2\):

P=((x₁+r x₂)/(1+r),(y₁+r y₂)/(1+r))

Interpretación gráfica

La distancia surge de las variaciones horizontal y vertical.

Distancias y ángulos

Distancia punto-recta

Para r: Ax+By+C=0 y P₀(x₀,y₀):

d=|Ax₀+By₀+C|/√(A²+B²)

Distancia dirigida

Sin valor absoluto, el signo indica el lado de la recta.

d_s=(Ax₀+By₀+C)/√(A²+B²)

Paralelismo

Dos rectas no verticales son paralelas si:

m₁=m₂

Perpendicularidad

Dos rectas no verticales son perpendiculares si:

m₁m₂=-1
